ESP Programmer 2
Advanced program for flashing and managing ESP8266 and ESP32 boards via a graphical interface.
Simple, lightweight, and with no external dependencies.
________________________________________

✨ Main features

•	Quick ESP8266 and ESP32 programming (multiple bin support).

•	Automatic detection of available serial ports.

•	Dedicated panel for each family (ESP32, ESP8266, Tools).

•	Automatic update check via GitHub on startup.

•	Detailed log of all operations.

•	Minimal and fast interface.
